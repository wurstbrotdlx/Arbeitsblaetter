Tippe eine Zutat an und tippe danach auf die passende Stufe der Pyramide. Wenn du fertig bist, drücke „Lösung anzeigen".

Zutaten

💡 Eine Kachel = eine Portion = eine Handvoll (pro Tag).
So geht's: 1️⃣ Zutat antippen → 2️⃣ Pyramiden-Stufe antippen. Tipp auf eine bereits einsortierte Zutat, um sie zurückzuholen.
